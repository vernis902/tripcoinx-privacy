Privacy Policy

Last updated: April 26, 2026

Overview

This Privacy Policy describes how TripCoinX ("the app", "we") handles information when you use our Android application. TripCoinX is a simple offline currency calculator for travelers.

Data We Collect

TripCoinX collects only the following data:

  • Currency settings — your selected source and target currencies and the exchange rate you have entered manually. Stored locally on your device only using Android SharedPreferences. Never transmitted to any server.
  • Anonymous feedback diagnostics — when you voluntarily submit a bug report or feature request through the in-app Help & Feedback screen, the following technical data is sent together with your message to help resolve issues faster:
    • App version name and code
    • Android API level
    • Selected currency pair (source → target)
    • Active theme (system / light / dark)
    • Device language tag (e.g. pl-PL)
    • Network type at submission time (WiFi / Cellular / Offline)
    • A randomly generated anonymous UUID (stored locally, never linked to your identity)

    This data is transmitted only when you explicitly tap "Send" in the feedback form. It does not include your name, email address, device model, location, or any other personally identifiable information. The diagnostic payload is processed by Make.com and used to create a private issue in the developer's bug tracker. It is not sold, shared with third parties, or used for advertising.

Third-Party Services

TripCoinX uses the following third-party services:

  • frankfurter.app — provides exchange rate data sourced from the European Central Bank (ECB). When the app refreshes rates in the background, it sends a standard HTTP GET request to api.frankfurter.app. The request contains no user data — only the base currency code (e.g. EUR). No API key or account is required. frankfurter.app is an open-source, public service; its source is available at github.com/frankfurter-app/frankfurter.
  • Google Play Billing Library — processes in-app purchases. When you make a purchase, Google processes the transaction on its own infrastructure. We do not receive or store your payment details. See Google's Privacy Policy.
  • Make.com (Integromat) — receives feedback reports submitted through the Help & Feedback feature. Data is limited to the anonymous diagnostic payload described above. Make.com's privacy notice is available at make.com/en/privacy-notice.

Data Sharing

We do not collect or sell personal information. The only data transmitted to third parties is:

  • Standard HTTP requests to frankfurter.app for exchange rate updates — contains no user data.
  • Anonymous diagnostic data submitted through the Help & Feedback form — only when you explicitly tap "Send", only to Make.com, as described above.

No data is ever sold, traded, or shared for advertising purposes.

Data Retention and Deletion

All local app data (currency settings and exchange rate) is stored only on your device. You can delete it at any time by:

  • Going to Settings → Apps → TripCoinX → Clear Data, or
  • Uninstalling the app

Children's Privacy

TripCoinX does not knowingly collect any data from children under the age of 13.

Changes to This Policy

We may update this Privacy Policy from time to time. The "Last updated" date at the top of this page will reflect any changes. Continued use of the app after changes constitutes acceptance of the updated policy.
